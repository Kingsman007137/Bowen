## 视觉规范文档 — 简约柔美版（毛玻璃 + 渐变色系）

### 1. 设计理念

* 风格定位：轻盈、柔和、漂浮感强，让用户在“记录思考”时感觉空间开阔、无压迫。
* 核心视觉：卡片采用毛玻璃（frosted glass）效果，背景模糊 + 半透明 +渐变叠加，传达“思绪漂浮”“点滴想法连接”的感觉。
* 色彩选择：避免高饱和强对比，采用低饱和渐变色＋黑白灰＋透明混合，营造柔美氛围。
* 亮／暗模式支持：用户可在“亮色系”与“暗色系”之间切换，以适应不同光线环境和使用偏好。

---

### 2. 色彩规范

#### 2.1 亮色系

* 主背景：从 `#F5F7FA` → `#FFFFFF`（非常浅灰渐变至白）
* 卡片毛玻璃底色：`rgba(255,255,255,0.70)` + 背景模糊
* 渐变色系预设：

  * 海雾蓝：`from #4E8CEB` → `to #64C7FF`
  * 柔粉金：`from #F6D2FF` → `to #FFE3D8`
  * 薄荷绿：`from #8EEFC4` → `to #56D9A0`
* 文字颜色：正文 `#111111`（深灰），标题 `#000000` 或 `#111111`
* 边框/分割线：`rgba(0,0,0,0.05)` 或 灰白色极浅线条

#### 2.2 暗色系

* 主背景：从 `#0F1115` → `#181C24`（深灰到更深）
* 卡片毛玻璃底色：`rgba(255,255,255,0.10)` + 背景模糊（或 `rgba(0,0,0,0.20)` 视情况）
* 渐变色系预设（暗调版）：

  * 海雾蓝-暗：`from #3B6BEC` → `to #4AC6FF`
  * 柔粉金-暗：`from #E8C0FF` → `to #FFD1C4`
  * 薄荷绿-暗：`from #6DD9AE` → `to #40C894`
* 文字颜色：正文 `#E6E6E6`（浅灰接近白），标题 `#FFFFFF` 或 `#F2F2F2`
* 边框/分割线：`rgba(255,255,255,0.20)` 或 极浅白线条

---

### 3. 材质与卡片规范

* 背景模糊：`backdrop-filter: blur(16–20px)`（亮模式略低模糊度，暗模式可略高）
* 卡片背景：毛玻璃底 + 渐变叠加。例如：

  ```
  background: linear-gradient(to bottom right, 起始色, 结束色);
  background: rgba(255,255,255,0.70);  /* 亮色系 */
  backdrop-filter: blur(16px);
  ```

  暗色系则 `rgba(255,255,255,0.10)` 或 `rgba(0,0,0,0.20)`。
* 圆角：统一 `16px`（或 Tailwind `rounded-2xl`）
* 阴影：

  * 亮色系卡片：`box-shadow: 0 6px 20px rgba(0,0,0,0.04)`
  * 暗色系卡片：`box-shadow: 0 8px 32px rgba(0,0,0,0.20)`
* 边框（可选描边）：`1px solid rgba(255,255,255,0.25)`（亮）或 `rgba(255,255,255,0.20)`（暗）
* 内边距：推荐 `padding: 1rem`（16px）或 `p-4`，可按需要加 `md:p-6`
* 最大宽度：约 `max-width: 400px`，保证卡片适度突出，不至于铺满画布太大

---

### 4. 排版与文字风格

* 字体：推荐使用 `Inter, system-ui, sans-serif`
* 字重：

  * 标题：`font-semibold` 或 `600`
  * 正文：`font-normal` 或 `400`
  * 引用/次要：`font-light italic`
* 字号：

  * H1（卡片标题）：20–24px
  * H2（卡片副标题）：18px
  * 正文（卡片内容）：14–16px
* 行高：1.4–1.6，确保阅读舒适
* 对比度：特别注意暗色系中正文与背景的对比要满足可读性标准（建议 `>= 4.5:1`）

---

### 5. 按钮／控件／交互状态

* 按钮或可点击控件建议使用“渐变强调色”（同卡片渐变色系之一）作为主色，与卡片风格统一但略区别。
* 悬浮状态（卡片或按钮）：

  * 透明度略提升
  * 阴影略加深（例如 `shadow-lg`）
  * 可微微放大 `scale(1.02)` 或使用 `transition: transform 200ms ease-in-out`
* 选中状态：

  * 卡片边框使用渐变描边（同当前背景渐变色系）
  * 或者背景色稍微加亮或加浓
* 切换主题（亮 ↔ 暗）：应有平滑过渡动画（例如 `transition: background-color 300ms ease, color 300ms ease`）

---

### 6. 背景与整体结构

* 画布背景：

  * 亮模式：渐变 `#F5F7FA` → `#FFFFFF`
  * 暗模式：渐变 `#0F1115` → `#181C24`
* 画布中的其他元素（如连线、辅助图形）颜色宜使用中性浅灰或带透明度的白色，保证不过分抢眼，但又能清晰区分。
* 卡片与背景预留足够间距、卡片之间有视觉“漂浮”感。避免信息堆叠过密。
* 保持整体界面简洁、轻盈，不要用太多色彩或纹理干扰，因为目标是“笔记”应用——用户花在内容而不是界面本身。

---

### 7. 实用 CSS / Tailwind 配置建议（可用于开发）

在 `tailwind.config.js` 中可以加入以下主题扩展：

```js
module.exports = {
  theme: {
    extend: {
      colors: {
        gradient: {
          blue: ['#4E8CEB', '#64C7FF'],
          pink: ['#F6D2FF', '#FFE3D8'],
          mint: ['#8EEFC4', '#56D9A0'],
          // 暗调版
          blueDark: ['#3B6BEC', '#4AC6FF'],
          pinkDark: ['#E8C0FF', '#FFD1C4'],
          mintDark: ['#6DD9AE', '#40C894'],
        },
        background: {
          light: '#F5F7FA',
          darkStart: '#0F1115',
          darkEnd: '#181C24',
        }
      },
      backdropBlur: {
        sm: '12px',
        md: '16px',
        lg: '20px',
      },
      boxShadow: {
        cardLight: '0 6px 20px rgba(0,0,0,0.04)',
        cardDark: '0 8px 32px rgba(0,0,0,0.20)',
      },
      borderRadius: {
        card: '16px',
      }
    },
  },
  variants: {
    backdropBlur: ['responsive'],
  },
  plugins: [],
}
```

卡片 CSS 可通过 className 类例如：

```html
<div class="rounded-card p-4 backdrop-blur-md bg-white/70 shadow-cardLight
     bg-gradient-to-br from-gradient-blue[0] to-gradient-blue[1]">
  <!-- 内容 -->
</div>
```

暗模式可用 `dark:` 前缀覆盖样式：

```html
<div class="dark:bg-white/10 dark:shadow-cardDark
     dark:from-gradient-blueDark[0] dark:to-gradient-blueDark[1]">
  <!-- 内容 -->
</div>
```

### 8. 可访问性 &易用性注意事项

* 虽然毛玻璃 + 半透明是美感关键，但必须保证**文字内容在卡片上仍具备良好可读性**。文字与背景的对比至少满足 WCAG AA 标准。
* 模糊和透明特效可能在低性能设备上影响渲染，建议提供“减少动画 /减少模糊”选项。
* 避免在高度交互区域（如表单输入、按钮）使用过度模糊或动画，因为可能降低可用性。
